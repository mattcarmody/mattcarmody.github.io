Shelved Projects

Trackr - Github repo

In the process of learning Python, I've been creating a "trackr" that monitors my day to day learning progress. It was entirely automated and all data was stored in a spreadsheet, showing me my quantified activity on sites I frequent for language learning, coding exercises and reading. I've since moved it to SQLite and added aspects of my physical activity, which require that I dictate an email with my activity to a dedicated email address. As I write this I'm in the process of creating automated visuals of my progress on a weekly, monthly, quarterly and annual basis, each with it's own scope that helps me plan the upcoming interval. The hope is to make it a web app and tie it into this site, though at this point I have no exposure to Python web frameworks and what wil be involved in that undertaking. (I'm using it as an opportunity to learn about Python web frameworks, not because they're necessarily the best tool for this job.) It could very well continue developing into a more extensive dashboard that pulls disparate elements of my life into one place and helps me stay on track.

Warm Up Quiz - Github repo

Though it was started first, this been largely sidelined by Trackr as far as where my attention goes lately. It's a bit of a meta project, a Python based script that quizzes me on Python concepts. It takes into account how well I understand each concept and when I last saw it, choosing quiz material accordingly. It's waiting for me to layout a more consistent set of data tables, complete the move from a spreadsheet to a database, potentially move from the terminal to a GUI, and expand the quiz material by scraping the Python docs. Once it is functional with a thorough set of Python material, it can be expanded to other topics.
