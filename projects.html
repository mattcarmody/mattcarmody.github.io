Slipstream

I'm one of three core Slipstream developers. I write core functionality of the CodeDog language, contribute to the Proteus engine and drive documentation for the project.

pyparsing, a Python package

pyparsing is a Python package for constructing a parser, used in ~140k GitHub repositories. I'm a member of the dev team, focused on contributing unit tests and documentation to the project.

Anki

Anki is a spaced repetition software (a memory tool) that I use every day. It's FOSS with an extensive community of add-ons that extend the base software. Recently, I've been contributing documentation for various add-ons. I'm also in the process of taking over as maintainer of an add-on.

Shelved Projects

Trackr

In the early days of learning Python, I wrote this script which monitors my day to day learning progress and shows me my progress. It runs on an automated schedule, scrapes my activity off the web, stores it in a SQLite database and generates data visualizations of my weekly and monthly progress.

Warm Up Quiz

It began as a bit of a meta project, a CLI Python script that quizzed me on Python concepts. It took into account how well I understood each concept and when I last saw it, choosing quiz material accordingly. After creating it I learned about spaced-repetition software and abandoned the project, adopting Anki in its place.
